# M30 SW Documentation

## 안녕하세요.:tada:



<mark style="color:blue;">**M30 SW Documentation에 오신걸 환영합니다!**</mark>&#x20;

M30 SW Documentation은 엔젤로보틱스의 로봇연구개발팀을 대상으로 엔젤로보틱스에서 제공하는 M30의 소프트웨어에 대한 상세한 가이드 메뉴얼을 목표로 하고 있습니다.

많은 기대 부탁드립니다 :clap:
